## 前言

傣族是我国南方一个具有悠久历史和独特文化的民族，其节日及民间故事丰富多彩。本项目旨在通过微信小程序推广傣族节日及民间故事，让更多人了解和认识傣族文化。以下是本项目相关内容的详细介绍。

## 内容介绍

本项目包含傣族的主要节日介绍、民间故事展示以及相关文化资讯。用户可以通过微信小程序随时随地了解傣族文化，感受傣族人民的热情与智慧。此外，我们还提供了丰富的互动功能，让用户可以参与到傣族文化的传承与推广中来。

## 技术介绍

本项目采用以下技术栈：

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.js 12/14/16
